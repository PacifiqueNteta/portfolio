Data Analytics Projects

I have completed several projects that have allowed me to hone my data analytics skills using various tools, including Excel (or Google Sheets), SQL, R, Power BI, and others..

Financial Analysis Project

This is a financial analysis project. The objective of the project is to gain valuable insights from a personal 6 months bank statement. I analyzed an person's bank statements from June 15, 2023, to January 15, 2024. I cleaned and categorized transactions in Excel, Power Query, and SQL Server. I then tracked key expenses and income metrics. Finally, I used Power BI to visualize these findings, creating a comprehensive report that highlighted key financial trends during this period.

Cleaning with Python

This is a data cleaning project performed on a coffee consumption survey dataset published by Maven Analytics as part of challenge. Details about the challenge and the dataset can be found on the following link https://mavenanalytics.io/challenges/maven-coffee-challenge/.

Data Exploratory Project

This project utilized Microsoft SQL Server to analyze Covid-19 data from 'Our World in Data'. I downloaded and preprocessed the data using Excel, separating it into two tables for loading into SQL Server. My analysis explored various aspects of the pandemic, including global and South African case counts, deaths, infection rates, and the dates of the first recorded cases and deaths.

Sales Excel Project - Interactive Dashboard

In this project, I created an interactive Dashboard in Microsoft Excel using sales data from a sample company. I cleaned data by removing duplicate then created pivot tables and charts to have an overview of the data and then assembled all charts to create and interactive dashboard
